# 注意力机制

## 核心思想

让模型在处理每个词时，"注意"到输入中其他相关词的信息。

## Scaled Dot-Product Attention

$$ \text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V $$

- **Q（Query）**：当前词的查询向量
- **K（Key）**：所有词的键向量
- **V（Value）**：所有词的值向量
- **$d_k$**：缩放因子，防止内积过大

## 多头注意力（Multi-Head Attention）

将 Q、K、V 拆分成多个头，每个头学习不同的注意力模式：

- 一个头关注语法关系
- 一个头关注语义相似性
- 一个头关注位置关系

最后将所有头的结果拼接起来。

## 自注意力 vs 交叉注意力

| 类型 | 说明 |
|------|------|
| **自注意力** | Q、K、V 来自同一序列（编码器内部） |
| **交叉注意力** | Q 来自一个序列，K、V 来自另一个（编码器-解码器之间） |

!!! tip "理解提示"
    可以把注意力机制理解为：模型在读到"它"时，自动去找到"它"指代的是哪个词，并把那个词的信息融合进来。
