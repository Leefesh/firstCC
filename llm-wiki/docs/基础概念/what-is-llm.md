# 什么是 LLM

## 定义

**LLM（Large Language Model，大语言模型）** 是一种基于海量文本数据训练而成的深度学习模型，能够理解和生成自然语言。

## 核心特点

- **大规模**：参数量通常在数十亿到数千亿级别
- **预训练**：在大规模无标注文本上学习语言规律
- **涌现能力**：当模型规模超过某个阈值后，出现小模型不具备的能力（如推理、代码生成）

## 典型代表

| 模型 | 开发者 | 特点 |
|------|--------|------|
| GPT-4o | OpenAI | 多模态，综合能力强 |
| Claude 4 | Anthropic | 长上下文，安全性好 |
| LLaMA 3 | Meta | 开源，可本地部署 |
| DeepSeek-V3 | 深度求索 | 性价比高，开源 |

## 局限

- 可能产生"幻觉"（生成不实信息）
- 知识有截止日期
- 缺乏真正的理解和推理能力
- 对输入敏感（Prompt 影响大）
